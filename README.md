# Wildlife AR Friend 2026

Wildlife AR Friend is an interactive Augmented Reality mobile application that allows users to learn about endangered Southeast Asian animals through AR interaction, feeding, facts, and bonding gameplay.

This project is designed as an educational AR experience for teenagers, helping them understand wildlife conservation in a more engaging and interactive way.

---

## Project Overview

The application lets users choose an endangered animal companion and place it in the real world using AR. After placing the animal, users can interact with it through feeding, playing, and learning facts.

The main goal of this project is to make wildlife education more fun, visual, and memorable through mobile AR technology.

---

## Featured Animals

- Indochinese Tiger
- Sumatran Elephant
- Lar Gibbon
- Malayan Tapir

Each animal has its own AR scene, interaction system, and educational content.

## How It Works

1. User opens the app.
2. User starts from the main menu.
3. User chooses an animal companion.
4. The app opens the selected AR scene.
5. User scans the environment.
6. User places the animal in AR.
7. User can feed, play, and learn facts about the animal.
8. Correct interactions increase the bond level and trigger positive feedback.

---

## Project Purpose

This project aims to raise awareness about endangered wildlife in Southeast Asia. Instead of only reading information, users can interact with animals directly in AR, making the learning experience more immersive and emotional.
