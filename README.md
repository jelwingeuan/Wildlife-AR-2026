# Wildlife AR Friend 🐾🌿

Wildlife AR Friend is an Android-based Augmented Reality edutainment app designed to teach kids and teenagers about endangered Southeast Asian animals through interactive AR companions.

Users can choose an endangered animal, place it in the real world using AR, and interact with it through simple actions such as feeding, playing, and learning facts.

## Project Overview

This project was created for **MMD6246 Immersive Media Design 4**.  
The assignment requires an AR application focused on endangered species, including their habitat, behaviour, and threat level. :contentReference[oaicite:0]{index=0}

## Target Users

- Kids and teenagers aged 12–15
- Students who want to learn about endangered wildlife in a fun and interactive way

## Selected Animals

Our group focuses on:

- Indochinese Tiger
- Sumatran Elephant
- Lar Gibbon
- Malayan Tapir

## Main Features

- AR animal placement using Unity AR Foundation
- Interactive animal companion
- Idle animation when the animal is not being interacted with
- Play interaction with animation feedback
- Educational facts about each endangered species
- Simple mobile UI for young users

## User Interaction Flow

1. User opens the app
2. User selects an animal companion
3. User scans a surface
4. The animal appears in AR
5. User can interact using buttons such as:
   - Play
   - Feed
   - Facts

## Tools & Software

- Unity
- AR Foundation
- Blender

## Project Goal

The goal of Wildlife AR Friend is to make endangered species education more engaging by combining AR technology, interactive animations, and child-friendly UI design.
